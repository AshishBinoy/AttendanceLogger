# AttendanceLogger
A Simple Go program to Log Attendance!

## Technology Stack
- Go

## How to Run
1. Make sure you have go (installed)[https://golang.org/doc/install].
2. Clone and navigate to this repository
3. Enter `go run main.go` in your command line
